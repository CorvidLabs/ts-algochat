ts-algochat

TypeScript implementation of AlgoChat - Encrypted messaging on Algorand

Features

End-to-End Encryption

X25519 + ChaCha20-Poly1305

Forward Secrecy

Per-message ephemeral keys

Bidirectional

Sender can decrypt own messages

Zero Dependencies

Uses @noble crypto (audited)

Installation

npm install ts-algochat

Quick Start

import { AlgorandService, createChatAccountFromMnemonic } from 'ts-algochat';

// Initialize service
const service = new AlgorandService({
    algodServer: 'https://testnet-api.algonode.cloud',
    indexerServer: 'https://testnet-idx.algonode.cloud',
});

// Create account
const account = createChatAccountFromMnemonic('your 25 word mnemonic...');

// Send encrypted message
